# OldRobloxStudioUIBringer
Easiest way to bring back old Roblox Studio UI design.

# How to use?
Download "OldRobloxStudioUIBringer.exe" file and open it to bring back the old Roblox Studio UI design.

# How it works?
It creates a folder inside of Roblox Studio that tells it to bring back the old UI.

# How to undo "OldRobloxStudioUIBringer.exe" actions?
1. Go to Studio installation folder, located at %appdata%\..\Local\Roblox\Versions\<version>. Make sure that there is RobloxStudioBeta.exe in the folder, otherwise, you are in a wrong place
2. Go to a folder named "ClientSettings".
3. Open the "ClientAppSettings.json" and change "FFlagEnableRibbonPlugin3" from "false" to "true".
4. Save the file and that's it.
